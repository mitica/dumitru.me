# dumitru.me

Personal blog rebuilt with Next.js + TypeScript.

## Content Structure

- `content/blog/*.md` for blog posts
- `content/projects/*.md` for project pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to `gh-pages`

```bash
npm run deploy
```

This runs a static export and pushes the generated `out/` content to the `gh-pages` branch.
